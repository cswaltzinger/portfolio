# Simple-Problems

This repository contains (mostly self contained) HTML documents that are desgined to solve one specific problem.  They are mostly intuitive to use and should be easily usable by the user.  

NOTE: most are desgined with the use of AI.  

### should_you_be_here.html
This is a simple html file that can act as a default response for people who attempt to access remote items that they are not allowed to.  

### bracket.html
This standalone html file allows the user to create a bracket and organize team matchups for virtually any competition.  

### wedding_seat_planner.html
This standalone html file is desgined to help my parents plan wedding seating arrangements.
