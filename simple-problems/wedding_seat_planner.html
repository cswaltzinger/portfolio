Wedding Seating Planner

Lay out rooms, arrange tables, and seat your guests
How to use this planner
  1. Create a room below with its width and length in feet.
  2. Add round or rectangular tables, then drag them anywhere inside the room.
  3. Click a table to select it, and use the +/- buttons on the right to set its size and number of chairs.
  4. Type each guest's name into a seat on the right, or click a chair in the drawing to jump straight to it.
  5. Use Export JSON to save your plan to a file, and Import JSON to load it again later or on another computer.
